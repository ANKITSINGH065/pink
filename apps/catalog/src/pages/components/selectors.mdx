# Selectors

### Choice selectors allow users to complete tasks which involve making choices such as selecting options, switching settings on or off, or adding or removing items.

## Types

In the Appwrite console, we use three types of choice selectors.

| Class    | Type     | Description                                                                                                   |
| -------- | -------- | ------------------------------------------------------------------------------------------------------------- |
| -        | Checkbox | Checkboxes allow users to select one or more items from a set.                                                |
| `switch` | Switch   | Toggles are used to quickly switch between two possible states. They are commonly used for “on/off” switches. |

<Preview center>
  <input type="checkbox" /> <input type="checkbox" checked />
  <input type="checkbox" class="switch" />
  <input type="checkbox" class="switch" checked />
</Preview>

## Radio

Radio buttons allow users to select one item from a set.

<Preview center>
  <input type="radio" name="radio" />
  <input type="radio" name="radio" checked="true" />
</Preview>

## States

In the Appwrite console, we use three types of choice selectors.

| Attribute  | Type     | Description                                       |
| ---------- | -------- | ------------------------------------------------- |
| `disabled` | Disabled | Shown when clicking on a selector is not allowed. |

<Preview center>
  <input type="checkbox" disabled /> <input type="checkbox" checked disabled />

  <input type="checkbox" class="switch" checked disabled />

## Size

There are three sizes of selectors; large, medium and small. Each size has its own purpose, so make sure you use every size correctly.

| class      | Type   | Description |
| ---------- | ------ | ----------- |
| `is-small` | Small  |             |
| -          | Medium |             |
| `is-large` | Large  |             |

## Usage

We recommend choice selectors next to labels should have at minimum 8px of space in between for legibility. Choice selectors should be aligned to the center of a single line of text. If there are multiple lines in a label should be aligned to the center of the first line in the label.

Checkboxes allow users to select multiple items from a set.

::::COMPARISON

![Icon with universal meaning](/images/button-do-1.svg)

![Specific icon and long sentence](/images/button-dont-1.png)

:::DO
Use checkboxes to allow users select one or more options from a list with related items.
:::

:::DONT
Do not use toggles, they don't imply the items are related, and take up more visual space.
:::

::::

Toggles and radio buttons refer to a single item. Radio buttons allow selecting only one item from a list, while toggles allow turning an item on or off.

::::COMPARISON

![Icon with universal meaning](/images/button-do-1.svg)

![Specific icon and long sentence](/images/button-dont-1.png)

:::DO
If each item in a set can be independently controlled, use toggles to allow on or off options.
:::

:::DONT
Do not use radio buttons to allow on or off options.
:::

::::
