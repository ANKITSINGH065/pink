# Shadows

Shadows are being use with CSS native variable

```html
<ul class="u-flex u-gap-12">
    <li>
        <div class="card" style="box-shadow:var(--shadow-small);">
            shadow-small
        </div>
    </li>
    <li>
        <div class="card" style="box-shadow:var(--shadow-large);">
            shadow-large
        </div>
    </li>
</ul>
```
