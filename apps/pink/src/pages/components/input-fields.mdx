# Input Fields

### Input fields provide a clear affordance for the user to enter and edit text, be easily discoverable, efficient and accessible. They typically appear in forms or dialogue modals.

## State

States can be applied to all inputs fields.

| Attribute  | Type     | Description |
| ---------- | -------- | ----------- |
| -          | Default  | TBW         |
| `disabled` | Disabled | TBW         |
| `readonly` | Readonly | TBW         |

## Helpers

`helper`

Helper text is information that assists the user in completing a input field.

| Class       | Type    | Description |
| ----------- | ------- | ----------- |
| -           | Default | TBW         |
| `u-success` | Success | TBW         |
| `u-warning` | Warning | TBW         |
| `u-error`   | Error   | TBW         |
