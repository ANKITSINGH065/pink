---
title: Icon Toggle
description: Icon toggle is used to switch between two possible states displayed with icons.
---

| Class            | Type          |                                             |
| ---------------- | ------------- | ------------------------------------------- |
| `icon-toggle`   | Icon Toggle    | A class representing an icon toggle.        |

<Preview center>
    <div class="icon-toggle">
        <ul class="icon-toggle-list">
            <li class="icon-toggle-item">
                <button class="icon-toggle-button" aria-label="List View">
                    <span class="icon-view-list" aria-hidden="true"></span>
                </button>
            </li>
            <li class="icon-toggle-item">
                <button class="icon-toggle-button is-selected" aria-label="Grid View">
                    <span class="icon-view-grid" aria-hidden="true"></span>
                </button>
            </li>
        </ul>
    </div>
</Preview>


## Components

Icons Toggle inner parts are:

| Class                | Type                                  |     |
| -------------------- | ------------------------------------- | --- |
| `icon-toggle-list`  | A class that wraps both toggle items  |     |
| `icon-toggle-item`  | A class that wraps one toggle item    |     |
| `icon-toggle-button`| An icon button                        |     |

##  States
To disable the toggle, add the following attribute:

| Attribute               | Type      |                                                  |
| ----------------------- | --------- | ------------------------------------------------ |
| `disabled`              | Disabled  | Use whenever clicking on a toggle isnâ€™t allowed. |
