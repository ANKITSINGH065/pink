# Cards

`card`

### Cards are used to group related content in a single container. This can include text, images, lists, buttons, and other elements.

## Styles

In the Appwrite console, we use two styles of cards borders:

| Class              | Type   |                                                         |
| ------------------ | ------ | ------------------------------------------------------- |
| -                  | Solid  | Solid border should be use in all cards.                |
| `is-border-dashed` | Dashed | We recommend you use dash border for empty state cards. |

## Usage

In Appwrite we are using cards to show empty states, to update / delete objects, to display overview, etc. A card can include avatars, buttons, illustrations, lists, tags and text for example:
