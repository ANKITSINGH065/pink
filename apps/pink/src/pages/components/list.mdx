## Types

In the Appwrite console we use two types of list items:

| Class               | Type              |                                                     |
| ------------------- | ----------------- | --------------------------------------------------- |
| `list-item`         | List Item         | Use in cases list items begin with an icon          |
| `numeric-list-item` | Numeric List Item | Use in cases list items begin with a number         |
