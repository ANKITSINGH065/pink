---
title: Action bar
description: The floating action bar is a tool that appears on top of the main layout and enables action execution.
---

| Class              | Type                  |                                              |
| ------------------ | --------------------- | -------------------------------------------- |
| `action-bar`       | Action Bar            | A class representing an action Bar container |
| `action-bar-start` | Action Bar Start side | Start side content of action Bar             |
| `action-bar-end`   | Action Bar End side   | End side content of action Bar               |
