---
title: Tooltip
description: Tooltips floating over the layout and provide additional information of user interface elements or features upon hover or focus.
---

| Class     | Type    |                                |
| --------- | ------- | ------------------------------ |
| `tooltip` | Tooltip | A class representing a tooltip |

## Components

Tooltips consist of two components:

| Class           | Type         |                                            |
| --------------- | ------------ | ------------------------------------------ |
| `tooltip-popup` | Text         |                                            |
| `is-bottom`     | Align Bottom | The tooltip will appear below the element. |
