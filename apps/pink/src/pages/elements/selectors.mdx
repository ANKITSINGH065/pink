## Variants

In the Appwrite console, we use three variants of choice selectors.

| Class    | Type     | Description                                                                                                    |
| -------- | -------- | -------------------------------------------------------------------------------------------------------------- |
| -        | Checkbox | Checkboxes allow users to select one or more items from a set.                                                 |
| `switch` | Checkbox | Switches are used to quickly toggle between two possible states. They are commonly used for “on/off” switches. |
| -        | Radio    | Radio buttons allow users to select one item from a set.                                                       |

## Usage

We recommend choice selectors next to labels should have at minimum 8px of space in between for legibility. Choice selectors should be aligned to the center of a single line of text. If there are multiple lines in a label should be aligned to the center of the first line in the label.

<br />

Checkboxes allow users to select multiple items from a set.

::::COMPARISON

![A list of users, with checkboxes alongside each one](/images/selectors-1.svg)

![A list of users, with switches alongside each one](/images/selectors-2.svg)

:::DO
Use checkboxes to allow users select one or more options from a list with related items.
:::

:::DONT
Do not use switches, they don't imply the items are related, and take up more visual space.
:::

::::

<br />

Switches and radio buttons refer to a single item. Radio buttons allow selecting only
one item from a list, while switches allow turning an item on or off.

::::COMPARISON

![A list of options ](/images/selectors-3.svg)

![](/images/selectors-4.svg)

:::DO
If each item in a set can be independently controlled, use switches to allow on or off options.
:::

:::DONT
Do not use radio buttons to allow on or off options.
:::

::::
