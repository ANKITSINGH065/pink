title: X Button
description: The x button is used inside other components, like modals or tables, for closing or removing items.
---
import CodePreview from "@components/CodePreview.svelte";

| Class            | Type           |                                                 |
| ---------------- | -------------- | ----------------------------------------------- |
| `x-button`       | Button         | A class representing x-button                   |

## Parameters

X button can get two params, using CSS variables:

| Variable                        | Value                                              |                                                |
| ------------------------------- | -------------------------------------------------- | ---------------------------------------------- |
| `--size`                        | 1.5rem (=24px)                                     | Dimensions (height and width)                  |
| `--button-text-color`           | --color-neutral-100/--color-neutral-50(light/dark) | Color (for both light and dark modes)          |
