---
title: Status
description: Process status indicates the current state of a complex process.
---

| Class            | Type    |                                                 |
| ---------------- | ------- | ----------------------------------------------- |
| `status`         | Status  | Status' representing class                      |


## Components

Status contains an icon:

| Class           | Type       |             |
| --------------- | ---------- | ----------- |
| `status-icon`   | Icon       |             |


## States

There are a few different states for Status:

| Class           | Type       |             |
| --------------- | ---------- | ----------- |
| -               | Default    |             |
| `is-warning`    | Waiting    |             |
| `is-pending`    | Pending    |             |
| `is-failed`     | Failed     |             |
| `is-complete`   | Complete   |             |
| `is-processing` | Processing |             |
