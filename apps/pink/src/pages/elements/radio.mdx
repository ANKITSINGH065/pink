### Size

There are three sizes of selectors; large, medium and small. Each size has its own purpose, so make sure you use every size correctly.

| class      | Type   | Description |
| ---------- | ------ | ----------- |
| `is-small` | Small  |             |
| -          | Medium |             |
| `is-large` | Large  |             |

<Preview center>

    <input type="radio" name="size" checked />
    <input type="radio" class="is-large" name="size" />
  </div>
</Preview>

## Usage

Switches and radio buttons refer to a single item. Radio buttons allow selecting only
one item from a list, while switches allow turning an item on or off.

::::COMPARISON

![A list of options ](/images/selectors-3.svg)

![](/images/selectors-4.svg)

:::DO
If each item in a set can be independently controlled, use switches to allow on or off options.
:::

:::DONT
Do not use radio buttons to allow on or off options.
:::

::::
