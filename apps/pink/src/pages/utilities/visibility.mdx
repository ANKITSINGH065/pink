---
title: Hide
---

Classes that handle visibility, opacity and overflow.

| Class                | Properties                      |
| -------------------- | ------------------------------- |
| `u-hide`             | `display: none !important;`     |
| `u-opacity-0`        | `opacity: 0 !important;`        |
| `u-opacity-0-2`      | `opacity: 0.2 !important;`      |
| `u-opacity-0-5`      | `opacity: 0.5 !important;`      |
| `u-overflow-hidden`  | `overflow: hidden !important;`  |
| `u-overflow-visible` | `overflow: visible !important;` |
| `u-overflow-y-auto`  | `overflow-y: auto !important;`  |
