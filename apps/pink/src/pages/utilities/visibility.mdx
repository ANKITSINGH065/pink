---
title: Visibility
description: An element's visibility, opacity and overflow are controlled by the classes below.
---

| Class                | Properties           |
| -------------------- | -------------------- |
| `u-hide`             | `display: nonet;`    |
| `u-opacity-0`        | `opacity: 0;`        |
| `u-opacity-20`       | `opacity: 0.2;`      |
| `u-opacity-50`       | `opacity: 0.5;`      |
| `u-overflow-hidden`  | `overflow: hidden;`  |
| `u-overflow-visible` | `overflow: visible;` |
| `u-overflow-x-auto`  | `overflow-x: auto;`  |
| `u-overflow-y-auto`  | `overflow-y: auto;`  |

<CodePreview>
  <div class="box u-overflow-y-auto">
    <p class="text">some text</p>
    <p class="u-hide">hide me</p>
  </div>
</CodePreview>
