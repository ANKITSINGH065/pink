---
title: Text
---

Classes that handle text size, boldness, line height and breaks

| Class                | Properties                                        |
| -------------------- | ------------------------------------------------- |
| `u-x-small`          | `font-size: pxToRem(12);`                         |
| `u-small`            | `font-size: pxToRem(14);`                         |
| `u-font-size-32`     | `font-size: pxToRem(32);`                         |
| `u-bold`             | `font-weight: 600;`                               |
| `u-normal`           | `font-weight: normal;`                            |
| `u-text-center`      | `text-align: center;`                             |
| `u-line-height-1`    | `line-height: 1;`                                 |
| `u-line-height-1-25` | `line-height: 1.25;`                              |
| `u-line-height-1-5`  | `line-height: 1.5;`                               |
| `u-line-height-2`    | `line-height: 2;`                                 |
| `u-break-word`       | `@include break-word; white-space: initial;`      |
| `u-break-all`        | `@include break-all; white-space: initial;`       |
| `u-trim`             | `@include trim;`                                  |
| `u-trim-start`       | `@include trim; direction:rtl; text-align:right;` |
| `u-trim-1`           | `@include trim(1);`                               |
| `u-un-break-text`    | `white-space: nowrap !important;`                 |
| `u-capitalize`       | `text-transform: capitalize;`                     |
| `u-icon-small`       | `font-size: var(--icon-size-small);`              |
