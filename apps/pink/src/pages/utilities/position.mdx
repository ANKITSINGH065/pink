title: Position
description: An element's position can be quickly configured by using Position classes.
---

| Class                   | Properties                       |
| ----------------------- | -------------------------------- |
| `u-position-absolute`   | `position: absolute;`            |
| `u-inset-inline-end-8`  | `inset-inline-end: pxToRem(8);`  |
| `u-inset-block-start-8` | `inset-block-start: pxToRem(8);` |
