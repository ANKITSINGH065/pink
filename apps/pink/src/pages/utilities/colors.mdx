## States Colors

In the Appwrite console we use six classes to set system colors based on states:

| Class        | Properties                                |
| ------------ | ----------------------------------------- |
| `u-disabled` | `color: hsl(var(--color-text-disabled));` |
| `u-offline`  | `color: hsl(var(--color-text-offline));`  |
| `u-info`     | `color: hsl(var(--color-text-info));`     |
| `u-error`    | `color: hsl(var(--color-text-danger));`   |
| `u-warning`  | `color: hsl(var(--color-text-warning));`  |
| `u-success`  | `color: hsl(var(--color-text-success));`  |

## Text Colors

In the Appwrite console we use a utility class to set our default text color:

| Class                     | Properties                            |
| ------------------------- | ------------------------------------- |
| `u-text-color-light-gray` | `color: hsl(var(--color-text-gray));` |
