---
title: Reset Inputs
description: A numeric input field's arrows are hidden when the reset inputs utility is used.
---

| Class                           | styles                                  |
| ------------------------------- | --------------------------------------- |
| `u-remove-input-number-buttons` | Chrome, Safari, Edge, Opera             |
|                                 | ` -webkit-appearance: none; margin: 0;` |
|                                 | Firefox                                 |
|                                 | ` -moz-appearance: textfield;`          |
