---
title: Responsive
---

Responsive global utility classes used to show/hide elements according to screen size.

| Class              | Properties                                                        |
| ------------------ | ----------------------------------------------------------------- |
| `.is-only-mobile`  | `@media #{$break2open} { display: none !important; }`             |
| `.is-only-tablet`  | `@media #{$break1}, #{$break3open} { display: none !important; }` |
| `.is-only-desktop` | `@media #{$break1}, #{$break2} { display: none !important; }`     |
| `.is-no-mobile`    | `@media #{$break1} { display: none !important; }`                 |
| `.is-no-desktop`   | `@media #{$break3open} { display: none !important; }`             |
