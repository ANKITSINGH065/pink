# Display

| -------------------------- | ----------------------------------------------------------------- |
| `u-block`                  | `display: block;`                                                 |
| `u-inline`                 | `display: inline;`                                                |
| `u-grid`                   | `display: grid;`                                                  |
| `u-flex`                   | `display: flex;`                                                  |
| `u-inline-flex`            | `display: inline-flex;`                                           |
|                            |                                                                   |
| `u-box-sizing-content`     | `box-sizing: content-box;`                                        |
| `u-sep-inline-start`       | `border-inline-start: solid pxToRem(1) hsl(var(--color-border));` |
| `u-sep-block-end`          | `border-block-end: solid pxToRem(1) hsl(var(--color-border));`    |
|                            |                                                                   |
| `u-gap-4`                  | `gap: pxToRem(4);`                                                |
| `u-gap-8`                  | `gap: pxToRem(8);`                                                |
| `u-gap-12`                 | `gap: pxToRem(12);`                                               |

|                            |                                                                   |
| `u-flex-vertical`          | `flex-direction: column;`                                         |

## Alignments

Classes that handle the aligment of the content

| Class                  | Properties                        |
| ---------------------- | --------------------------------- |
| `u-main-center`        | `justify-content: center;`        |
| `u-main-space-between` | `justify-content: space-between;` |
| `u-main-end`           | `justify-content: end;`           |
|                        |                                   |
| `u-cross-center`       | `align-items: center;`            |
| `u-cross-end`          | `align-items: end;`               |
|                        |                                   |
| `u-cross-child-start`  | `align-self: start;`              |
| `u-cross-child-center` | `align-self: center;`             |
| `u-cross-child-end`    | `align-self: end;`                |
