# Grid box

Create items in a row according to container width

# Grid box with tiles

Can Include Tiles

```html
<ul class="grid-box">
  <li class="tile">
    <a class="card" href="">
      <h2 class="tile-title">Tile Name</h2>
      <span class="icon-right-open" aria-hidden="true"></span>
    </a>
  </li>
  <li class="tile">
    <a class="card" href="">
      <h2 class="tile-title">Tile Name</h2>
      <span class="icon-right-open" aria-hidden="true"></span>
    </a>
  </li>
  <li class="tile">
    <a class="card" href="">
      <h2 class="tile-title">Tile Name</h2>
      <span class="icon-right-open" aria-hidden="true"></span>
    </a>
  </li>
</ul>
```
